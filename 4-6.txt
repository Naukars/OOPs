Q.4 What is DataType?

A data type tells Java what kind of value is stored and what operations are valid, ensuring type safety at compile time. 
Common categories are primitive types and reference types.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------
Q.5 Different Types of Data Types.

Primitive types are the basic building blocks like int, double, boolean, and char; they store actual values and live on the stack. 
Reference types store references to objects (arrays, classes, interfaces) on the heap.
